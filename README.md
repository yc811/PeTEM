# System requirements
### R environment:
*	R version ≥ 4.2 (tested on 4.3.2)
*	Required R packages:
    * optparse
    * dplyr
    * tidyr
    * zoo
    * reshape2
    * stringr
    * ggplot2
    * gplots
    * ggalluvial
    * ggpointdensity
    * RColorBrewer
    * viridis

### Python environment:
*	Python ≥ 3.8 (tested on 3.8.10)
*	Required Python packages:
    * pandas (≥ 1.2.4)
    * (uses built-in: glob, os, time)

### Bioinformatics tools:
*	samtools (tested on 1.10)
*	bedtools (tested on v2.27.1)
